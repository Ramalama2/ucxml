Accounts opted as admin have special privileges.

Upon selecting Site Administration, you will be shown a 4 column table containing all of the user accounts.
  • Username - User's login name for this web interface
  • Email - User's email address
  • Account Type - Assigns Account privileges for this user web login.
    • Admin - Has access to all parts of the web interface.
    • User - Has limited privileges Some pages is denied.
  • Edit/Delete - Clicking edit will direct you to the Edit User page. Clicking delete will delete this user from the database.
